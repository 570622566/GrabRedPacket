# GrabRedPacket

一款智能抢红包App，基于Android手机的`AccessibilityService`辅助服务功能，实现`微信端`、`QQ端`自动抢红包功能，黑屏锁屏照样抢。并加入多个设置功能，如抢红包群过滤功能、时间段设置功能、延迟设置功能等等
<br>
**注：由于QQ更新版本后，通知栏提示信息有所变动，导致QQ端抢红包功能失效。**
